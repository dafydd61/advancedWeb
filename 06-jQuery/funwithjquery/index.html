But wait, there's more…

The example above used JavaScript/jQuery to show and hide content, which is great, but in some cases (unfortunately, not the one above!) there's a more elegant way to pull this off.

This time, we're using jQuery just to add and remove a class name ('reveal-is-visible'). We let CSS3 deal with the visibility, using timed transitions.

Note that this will fall back nicely in older browsers - you won't see the transition, but the content will still show and hide.
